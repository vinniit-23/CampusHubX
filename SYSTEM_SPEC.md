Product: CampusHubX (CampusConnect+ Ecosystem)
Type: Full-stack web application
Users: Students, Colleges, Recruiters

Frontend:
- React + Vite
- TailwindCSS
- Role-based dashboards

Backend:
- Node.js + Express
- JWT authentication
- REST API

Database:
- MongoDB (Mongoose)

Core Modules:
1. Auth & Roles
2. Student Dashboard
3. Skill & Project Tracker
4. Opportunity Matching Engine
5. Verified Achievements
6. Recruiter Dashboard
7. Analytics (basic)

Deployment:
- Frontend: Vercel
- Backend: Render
- DB: MongoDB Atlas
